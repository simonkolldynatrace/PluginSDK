Known process types list
========================
Processes are deprecated. Please use technologies for plugin activation.

The table presents names and IDs of known process types supported by Dynatrace, which are used in metadata section of plugin.json.

Use `Process type name` in `processTypeNames` and `Process type ID` in `processTypes`.
